INTRODUCTION

The project cityservices tests the knowledge base located into the website Dydra. It is a complex code, written in javascript, where you can find an open dataset for macrostructure of Comune di Catania that are modeled following the semantic web. It's also possible to do researches of the dishes with the navigation bar named "Query". If you select a specific dish, a nationality or both, you will see a dishes list with the selected characteristics.

Open Data

The municipality of Catania provides a dataset Farmacie which contains all offices of the city. This dataset is released with no licensing indications, so that it can be considered as released as Public Domain.

Such a CSV file, provided by the municipality of Catania, has been turned into an OWL ontology (in which the LOCN vocabulary is used) by the csvCatania2PharmaciesOWL.php script (see the Git repository). This script is released under the LGPL 3.0 license, whereas the produced ontology mantains the CC0 license of the origin dataset.

farmacie.owl

Tests on pharmacies has been reported in a different ontology, linked with the former, using the EARL vocabulary. This ontology is released under the CC BY 4.0 license.

comunect.owl

The farmacie.owl has been then enriched by using the reasoner Pellet (version 2.3.0). Such an enriched ontology, as well as the comunect.owl one, are provided together via a sparql endpoint gracefully provided by Dydra.
