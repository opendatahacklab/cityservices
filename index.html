INTRODUCTION

The project cityservices is a visualization tool in javascript for public services and organizational structures modeled via the Core Public Service Vocabulary. The implementation presented here uses the knowledge base about the municipality of Catania described in the page Open Data Unofficial del Comune di Catania. Note that the ontology is outdated as the organizational structure of the municipality of Catania changed repeatedly from the ontology creation time.
